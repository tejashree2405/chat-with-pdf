# Chat With PDF

A Retrieval-Augmented Generation (RAG) application that allows users to upload one or more PDFs and ask questions about their contents.

## Features

- Multi-PDF Upload
- Groq LLM Integration
- ChromaDB Vector Store
- Semantic Search
- Source Citations
- FastAPI Backend
- Streamlit Frontend

## Tech Stack

- FastAPI
- Streamlit
- LangChain
- ChromaDB
- Groq
- HuggingFace Embeddings
